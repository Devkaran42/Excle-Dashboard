## Excle-Dashboard

#Sales Analysis Dashboard (Excel)
This project is an interactive Sales Analysis Dashboard built in Microsoft Excel to analyze and visualize sales performance data efficiently.
## Dataset used
-<a herf="https://github.com/Devkaran42/Excle-Dashboard/blob/main/Sales%20Analysis%20Dashboard%20Practice.xlsx">Dataset view</a>

#Key Features
Dynamic dashboard with clear data visualization
Sales performance tracking (revenue, profit, quantity)
Region-wise and product-wise analysis
Interactive filters (slicers) for easy data exploration
Clean and user-friendly layout
